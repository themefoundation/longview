# Longview
Longview is a WordPress theme created specifically for developers.

## Features

### Layouts

* Content Only (standard width)
* Full width content
* Content/Sidebar
* Content/Sidebar/Sidebar
* Sidebar/Content
* Sidebar/Sidebar/Content
* Sidebar/Content/Sidebar

### Widget areas

* Before Header
* Header
* After Header
* Sidebar
* Additional Sidebar
* Before Footer
* Footer
* After Footer
